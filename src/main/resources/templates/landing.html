About the Project

This dashboard integrates environmental (temperature, humidity, precipitation) and inspection (colony size, brood frames, queen status) data from the UrBAN dataset to provide urban beekeepers with clear, actionable insights. By visualizing trends and health scores across multiple hives, users can optimize hive management, improve honey yield, and react promptly to critical conditions.

Features

Real-Time Monitoring

Live tracking of hive temperature, humidity, and precipitation.

Insights

Benchmark hive health scores and rank colonies

Reporting & Export

Download data snapshots in CSV for record-keeping.

Technology

Java & Spring Boot

Robust backend REST API and data processing engine.

Thymeleaf Templates

Server-rendered views with dynamic data binding.

Chart.js

Interactive, responsive charts for data visualization.

MIT-Licensed UrBAN Dataset

Urban Beehive Acoustics & Phenotyping environmental data.
